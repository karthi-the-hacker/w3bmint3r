# w3bmint3r
used to find webmin service

w3bmint3r tool Installation steps:

1. cd ~
2. mkdir tools
3. cd ~/tools
4. git clone https://github.com/karthi-the-hacker/w3bmint3r.git
5. cd w3bmint3r
6. chmod +x w3bmint3r
7. chmod +x .banner
8. sudo ./w3bmint3r
